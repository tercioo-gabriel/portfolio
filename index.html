Habilidades

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar os elementos da página (textos, links, imagens...). É a base das páginas web.

CSS (Cascading Style Sheets) é uma linguagem que controla a aparência visual de páginas da web. Permite definir cores, fontes, layout e mais. Essas regras são aplicadas a elementos HTML usando seletores e declarações.

JavaScript é a principal linguagem de programação usada para desenvolvimento web. Ela é muito utilizada para manipulação de elementos HTML e CSS (DOM), requisições assíncronas (AJAX) e responder a eventos.

TypeScript é um "superset" da linguagem JavaScript, adiciona um sistema de tipagem estática e recursos avançados de linguagem, tornando o desenvolvimento mais robusto, eficiente e seguro.

Photoshop e Illustrator são dois programas de software desenvolvidos pela Adobe, voltados para edição de imagens e criação de gráficos vetoriais, respectivamente. Amplamente utilizados no Design Gráfico e UI/UX Design.

Git é um sistema de controle de versão distribuído, projetado para acompanhar mudanças em arquivos ao longo do tempo e facilitar a colaboração entre várias pessoas em projetos de desenvolvimento de software. Indispensável na programação.
