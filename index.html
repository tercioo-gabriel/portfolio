Habilidades

Passe o cursor do mouse ou clique por cima das habilidades

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar os elementos da página (textos, links, imagens...). É a base das páginas web.

CSS (Cascading Style Sheets) é uma linguagem que controla a aparência visual de páginas da web. Permite definir cores, fontes, layout e mais. Essas regras são aplicadas a elementos HTML usando seletores e declarações.

JavaScript é a principal linguagem de programação usada para desenvolvimento web. Muito utilizada para manipulação de elementos HTML e CSS (DOM), requisições assíncronas (AJAX) e responder a eventos.

TypeScript é um "superset" da linguagem JavaScript, adiciona um sistema de tipagem estática e recursos avançados de linguagem, tornando o desenvolvimento mais robusto, eficiente e seguro.

React é uma poderosa biblioteca JavaScript usada para construir interfaces de usuário interativas, eficientes e reutilizáveis em aplicações web modernas de alta qualidade.

Styled-Components é uma biblioteca JavaScript para web. Ela permite que você crie estilos CSS de forma mais modular e dinâmica, associando estilos diretamente aos seus componentes React.

MySQL é um sistema de gerenciamento de banco de dados relacional (RDBMS) de código aberto usado para armazenar e recuperar dados de maneira estruturada usando SQL.

Git é um sistema de controle de versão distribuído, projetado para acompanhar mudanças em arquivos ao longo do tempo e facilitar a colaboração entre as pessoas em um ambiente de desenvolvimento de software.
